T3 혹한기·혹서기 작전가용일 산출 로직 LOGIC 부대관리훈령 기준 적용: 혹서기 = WBGT 31°C 이상 일수 / 혹한기 = 최저기온 -15°C 이하 일수. 월별 평균 가용일 수 = 총일수 - (불가일 수). 10년 서브기간(1991-2000 / 2001-2010 / 2011-2020) 트렌드 비교 계급별일수 데이터 파생 WMO 기후평년값의 10년 서브기간 이미 제공 → 기후변화 트렌드 직접 활용 가능
T4 월별 바 차트 시각화 UI X축: 1~12월 / Y축: 작전가용일 수. 올해 실측(파란 막대) vs 10년 평년(회색 막대) 쌍대 비교. 혹한기(12-2월) / 혹서기(7-8월) 구간 배경 하이라이트. plotly 또는 altair 권장 - 최적 훈련 시즌(봄·가을) 자동 표시 추가
T5 이상기후 지표 요약 카드 UI "올해 연간 작전가용일 XX일 (평년 대비 ±N일)" 요약 카드. 기온·강수·황사·폭염일수 편차 게이지 바 표시. 이상기후 판정 시 경고 배너 자동 표시 - st.metric + st.progress 조합으로 Streamlit 구현 가능
한국기후표 활용 전략: 직접 운용 판단에 쓰지 말고 "비교 기준선 DB"로만 활용. 월평균 기온 대신 계급별일수(Ⅱ-437 섹션) 추출이 가용일수 산출 정확도의 핵심
